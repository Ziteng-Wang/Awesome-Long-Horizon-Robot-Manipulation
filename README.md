## Definition

Long-horizon robot manipulation refers to manipulation tasks where success depends on executing a sequence of interdependent subtasks under language, multimodal, or open-ended goals. The key difficulty is not only sequence length, but the need to maintain task memory, progress state, object-state changes, failure signals, and replanning context during execution.

## Taxonomy

### Architecture Layers

- **Classical planning, TAMP, and HRL**: STRIPS, PDDL, task-and-motion planning, options, skill abstraction, and hierarchical control.
- **LLM/VLM planner plus skill library**: semantic task decomposition, affordance grounding, program generation, and skill invocation.
- **Language-conditioned and prompt-conditioned policies**: direct mapping from language or multimodal prompts to manipulation actions.
- **VLA and robot foundation models**: large-scale vision-language-action models trained on cross-task and cross-embodiment robot data.
- **Generative action policies**: diffusion policies, flow matching, action chunking, video subgoals, and 3D/action-conditioned future prediction.
- **Hybrid long-horizon agents**: closed-loop systems combining planner, memory, progress estimator, VLA/policy, world model, verifier, and recovery.

### Long-Horizon Mechanisms

- **Temporal abstraction**: subgoal decomposition, skill selection, option termination, task graphs.
- **Task memory**: working memory, episodic memory, semantic memory, object-centric memory, retrieval-augmented policy state.
- **Progress estimation**: subtask completion prediction, phase awareness, transition timing.
- **Failure detection and recovery**: success classifiers, verifiers, rollback, replanning, retry policies.
- **World-model foresight**: action-conditioned future prediction, imagined states, visual subgoals, physical consistency checks.
- **Safety and constraints**: constraint-aware planning, unsafe-action detection, human correction and preference alignment.

## Representative Methods

### Planning and Skill Composition

- [SayCan](https://say-can.github.io/) - grounding language plans with affordance/value functions.
- [Inner Monologue](https://innermonologue.github.io/) - closed-loop language feedback for embodied planning.
- [Code as Policies](https://code-as-policies.github.io/) - programmatic robot policies generated by language models.
- [ProgPrompt](https://progprompt.github.io/) - situated task plans as executable programs.
- [VoxPoser](https://voxposer.github.io/) - language-conditioned 3D value maps for robotic manipulation.
- [LISA](https://arxiv.org/abs/2203.00054) - interpretable skill abstractions learned from language.

### Language-Conditioned and Prompt Policies

- [CLIPort](https://cliport.github.io/) - language-conditioned pick-and-place with CLIP and Transporter Networks.
- [PerAct](https://peract.github.io/) - multi-task manipulation with voxelized 3D representations.
- [Hiveformer](https://arxiv.org/abs/2209.04899) - history-aware instruction-following manipulation.
- [RVT](https://robotic-view-transformer.github.io/) - multi-view transformer for 3D object manipulation.
- [VIMA](https://vimalabs.github.io/) - multimodal prompt-conditioned robot manipulation.

### VLA and Robot Foundation Models

- [RT-1](https://robotics-transformer1.github.io/) - real-world robotic Transformer at scale.
- [RT-2](https://robotics-transformer2.github.io/) - transferring web knowledge to robotic control.
- [PaLM-E](https://palm-e.github.io/) - embodied multimodal language model.
- [Open X-Embodiment / RT-X](https://robotics-transformer-x.github.io/) - cross-embodiment robot learning datasets and models.
- [Octo](https://octo-models.github.io/) - open-source generalist robot policy.
- [OpenVLA](https://openvla.github.io/) - open-source 7B VLA trained on large robot demonstrations.
- [Pi0](https://www.physicalintelligence.company/blog/pi0) - VLA flow model for general robot control.
- [Pi0.5](https://www.physicalintelligence.company/research/knowledge_insulation) - open-world generalization in VLA control.
- [RDT-1B](https://rdt-robotics.github.io/rdt-robotics/) - diffusion foundation model for bimanual manipulation.

## Benchmarks and Datasets

### Foundational Manipulation Benchmarks

- [Meta-World](https://meta-world.github.io/) - multi-task and meta-RL manipulation.
- [RLBench](https://sites.google.com/view/rlbench) - language-conditioned manipulation benchmark.
- [ManiSkill2](https://maniskill2.github.io/) - generalizable manipulation skills.
- [RoboMimic](https://robomimic.github.io/) - imitation learning benchmark from demonstrations.

### Long-Horizon and Language Benchmarks

- [CALVIN](http://calvin.cs.uni-freiburg.de/) - language-conditioned long-horizon manipulation.
- [VIMA-Bench](https://vimalabs.github.io/) - multimodal prompt manipulation.
- [LIBERO](https://libero-project.github.io/) - lifelong robot learning, including LIBERO-Long.
- [FurnitureBench](https://clvrai.github.io/furniture-bench/) - real and simulated furniture assembly.

### Real-World and Cross-Embodiment Data

- [Open X-Embodiment](https://robotics-transformer-x.github.io/) - cross-embodiment robot dataset.
- [DROID](https://droid-dataset.github.io/) - in-the-wild real robot manipulation data.
- [BridgeData V2](https://rail-berkeley.github.io/bridgedata/) - large-scale robot learning dataset.
- [Mobile ALOHA](https://mobile-aloha.github.io/) - whole-body bimanual mobile manipulation.

### Household and Diagnostic Evaluation

- [RoboCasa](https://robocasa.ai/) - kitchen scenes and everyday household manipulation.
- [RoboCasa365](https://arxiv.org/abs/2603.04356) - large-scale household simulation for generalist robots.
- [BEHAVIOR-1K](https://behavior.stanford.edu/) - 1,000 everyday household activities.
- [VLABench](https://arxiv.org/abs/2412.18194) - long-horizon reasoning tasks for language-conditioned manipulation.
- [RMBench](https://arxiv.org/abs/2603.01229) - memory-dependent manipulation evaluation.
- [LongAct](https://arxiv.org/abs/2605.14504) - free-form long-horizon household task execution.

## Reported Results

Results below are reported by original papers and are not directly comparable across protocols, robots, datasets, or backbones.

| Method | Benchmark | Reported headline result | Mechanism |
|---|---|---|---|
| OpenVLA | 29 robot tasks | +16.5 points over RT-2-X; +20.4 points over Diffusion Policy in reported settings | Open VLA scaling and fine-tuning |
| PALM | LIBERO-LONG, CALVIN ABC-to-D, real | 91.8% on LIBERO-LONG; 12.5% average-length improvement on CALVIN; 2x real-world improvement | Affordance and progress |
| MemoryVLA | LIBERO, SimplerEnv, Mikasa-Robo, real | 96.5% on LIBERO-5 suites; 84.0% on 12 real tasks; +26 points on real long-horizon tasks | Perceptual-cognitive memory |
| MemoryVLA++ | LIBERO, SimplerEnv, Mikasa-Robo, CALVIN, real | +9 / +26 / +28 point real-robot gains on general, memory-dependent, and imagination-dependent tasks | Memory plus imagination |
| Long-VLA | L-CALVIN, real | Reports new SOTA on long-horizon L-CALVIN chains | Phase-aware VLA |
| HELM | LIBERO, CALVIN | Reports reduced memory, verification, and recovery gaps | Memory, verifier, rollback/replan |
